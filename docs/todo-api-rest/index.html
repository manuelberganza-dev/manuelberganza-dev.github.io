Todo App - API REST

Spring MVC no solo tiene un gran potencial para crear aplicaciones web renderizadas en el servidor, sino también para crear API REST siguiendo este patrón.

Tecnologías: Spring MVC, Spring Data JPA, Spring Security, Spring Validation, MySQL

Diseñada para gestionar tareas y listas de pendientes de forma eficiente, permitiendo a los usuarios organizar sus actividades diarias de manera estructurada y segura.

La API proporciona funcionalidades esenciales para crear, actualizar y eliminar tareas, además de poder asignar prioridades y categorías a cada una de ellas, ayudando a los usuarios a mantener un control preciso sobre sus pendientes.

Permitiendo la integración con interfaces frontend que acceden a la API de forma segura.
