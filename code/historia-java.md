<h2 class="text-center text-4xl font-bold mb-8">Historia del Lenguaje Java</h2>

<img class="w-full max-w-xl mx-auto rounded-md mb-10" src="/img/post/java-portada.jpg" />

<div class="mb-6 space-y-6">
    <p>
        Java es uno de los lenguajes de programación más populares, su historia inicia a principios de los 90s en Sun Microsystem. Con el proyecto llamado <span class="text-mb-blue-light"><strong>"The Green Project"</strong></span>, liderado por <span class="text-mb-blue-light"><strong>James Gosling</strong></span> y otros ingenieros, cuyo objetivo era crear un lenguaje para multiples dispositivos, eficiente y seguro.
    </p>

    <div class="w-fit mx-auto">
        <img class="w-72 rounded-md" src="/img/post/james-gosling.jpg">
        <h2 class="text-center font-bold text-xl mt-2">James Gosling</h2>
    </div>

    <p>
        Inicialmente, el lenguaje fue conocido como <span class="text-mb-blue-light"><strong>"Oak"</strong></span> y estaba destinado a dispositivos electrónicos, pero cuando la <span class="text-mb-blue-light"><strong>World Wide Web</strong></span> comenzó a ganar popularidad, los desarrolladores vieron una oportunidad para utilizarlo en aplicaciones web. Finalmente, fue renombrado como <span class="text-mb-blue-light"><strong>"Java"</strong></span> en 1995 debido a problemas de derecho de autor con el nombre <span class="text-mb-blue-light"><strong>"Oak"</strong></span>.
    </p>

    <p>
        Desde su creación, Java ha seguido evolucionando. Con <span class="text-mb-blue-light"><strong>Oracle</strong></span> a cargo de su desarrollo desde 2010, Java ha adoptado un ciclo de lanzamientos más rápido, con <span class="text-mb-blue-light"><strong>actualizaciones</strong></span> cada seis meses, lo cual ha permitido que las nuevas funcionalidades lleguen de manera más oportuna. Este enfoque ha ayudado a Java a adaptarse mejor al mercado.
    </p>

    <div class="w-fit mx-auto">
        <img class="w-72 rounded-md" src="/img/tech/oracle.png">
    </div>

</div>

<div class="mb-6 space-y-6">
    <p>
        Java alcanzó una gran popularidad gracias a su lema: <span class="text-mb-blue-light"><strong>“escribe una vez, ejecuta en cualquier lugar”</strong></span>. Lo cual se hizo posible por el uso de la <span class="text-mb-blue-light"><strong>Java Virtual Machine (JVM)</strong></span>. Este enfoque permitía que el código Java compilado se ejecutara en cualquier sistema que tuviera la <span class="text-mb-blue-light"><strong>JVM</strong></span> instalada, independientemente de la arquitectura de hardware o el sistema operativo.
    </p>

    <p>
        Hoy en día Java es fundamental en el desarrollo de <span class="text-mb-blue-light"><strong>aplicaciones empresariales</strong></span>, <span class="text-mb-blue-light"><strong>aplicaciones móviles</strong></span>, y <span class="text-mb-blue-light"><strong>Sistemas backend complejos</strong></span>. La comunidad es muy activa y gracias a su estructura de código abierto, facilitada por el proyecto <span class="text-mb-blue-light"><strong>OpenJDK</strong></span>, han sido clave para su longevidad y éxito que muy pocos lenguajes logran permanecer relevantes a lo largo del tiempo.
    </p>

    </div>
</div>
